🎨 CSS (Cascading Style Sheets)

HTML로 만든 구조를 **시각적으로 꾸미는** 스타일 시트 언어.

핵심 문법

CSS는 `선택자 { 속성: 값; }`의 형태로 작성됩니다.

.box {
    width: 200px;
    height: 100px;
    background-color: var(--color-accent-pink); /* 파스텔 핑크 */
    border-radius: 5px;
}

선택자의 종류

  • **클래스 선택자:** `.classname` (가장 많이 사용하며 유연함)
  • **ID 선택자:** `#idname` (페이지에서 유일한 요소에 적용)
  • **요소 선택자:** `p`, `div` (해당 태그 전체에 적용)

🚀 JavaScript

웹 페이지에 **동작, 이벤트, 상호작용**을 부여하는 프로그래밍 언어.

주요 기능

  • **DOM (Document Object Model) 조작:** HTML 요소를 **선택**하고 내용을 **변경**하거나 **새로 생성**할 수 있습니다.
  • **이벤트 처리:** 마우스 클릭, 키보드 입력 등 사용자 행동에 반응하는 코드를 실행합니다.
  • **비동기 통신 (AJAX/Fetch):** 서버와 데이터를 주고받아 페이지를 다시 로드하지 않고 정보를 업데이트합니다.
