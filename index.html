📌 HTML (HyperText Markup Language)

웹 페이지의 **구조**를 잡는 마크업 언어.

주요 개념

  • **태그(Tag):** `<요소명>내용` 형태로 사용되며, 웹 요소의 종류와 역할을 정의함.
  • **요소(Element):** 시작 태그와 종료 태그, 그리고 그 사이의 내용 전체를 의미함.
  • **속성(Attribute):** 태그에 추가 정보를 제공하며, ``처럼 사용됨.

📌 CSS (Cascading Style Sheets)

HTML로 만든 구조에 **스타일**을 입히는 언어.

선택자(Selector)

어떤 HTML 요소에 스타일을 적용할지 선택하는 방법.

  • **요소 선택자:** `p { ... }` (모든 `

    ` 태그)

  • **클래스 선택자:** `.main { ... }` (class="main"인 요소)
  • **ID 선택자:** `#header { ... }` (id="header"인 요소)

CSS 기본 문법

`선택자 { 속성: 값; 속성: 값; }` 형태

.box {
    width: 100px;
    height: 100px;
    background-color: blue;
}

📌 JavaScript

웹 페이지에 **동작과 상호작용**을 추가하는 프로그래밍 언어.

핵심 역할

  • **DOM 조작:** HTML 요소를 가져와서 내용을 바꾸거나 스타일을 변경함.
  • **이벤트 처리:** 버튼 클릭, 마우스 오버 등 사용자의 동작에 반응함.
  • **비동기 통신:** 서버와 데이터를 주고받음 (AJAX, Fetch API).
